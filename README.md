# IT-204
Birchwood Lane Schools DBMS Proposal 

The Birchwood Lane Schools are required to transition from their current paper records to an electronic records system in order to comply with state and federal regulations. Their current record system of student, instructor and course records is outdated, inefficient, and can be time-consuming on issues requiring quick feedback due to extensive search times. A paper system could imply data security issues from use of simple filing cabinets while also the added security of paper files being left in the open. This initiative includes digitizing existing paper files and establishing an efficient framework for managing all future records. Implementing a DBMS will offer Birchwood Lane Schools a modernized and efficient approach to record management

This proposal includes visualizations of a Basic Rational Database, Conceptual Data Model, and Logical Data Model.
